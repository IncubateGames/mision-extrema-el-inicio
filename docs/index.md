# Descargas
Version original del juego: [Mision extrema - Game](Mision extrema - Game.zip)
### Instrucciones
* Descargar el zip
* descomprimir en cualquier carpeta
* Ejecutar el archivo mision-extrema-game.cmd o ir a la carpeta Release y ejecutar em.exe
